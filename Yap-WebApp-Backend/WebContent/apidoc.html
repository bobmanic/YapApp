YAP API

Request :

/Yap/api?

Sample Request (Search) :

/Yap/api?app_id=11&request_type=search&form=strict&keywords=sta&city=San&state=C&type=restaurant

Request Parameters :

app_id - An authentication key provided to developers from the developer console. Type integer.

request_type - Provides two APIs; Reviews ("reviews") & Business Search ("search"). Type String.

form - Parameter for Business Search to be "lenient" or "strict", for all parameters to match exactly or search similar parameters. Type String.

keywords - Search string to be compared with business name . Type String.

city - City of Business. Type String.

state - State of Business. Type String.

 

type - Type of Business, defined only as "restaurant", "shop", "hotel", "cafe" and "bar".


 

Output Parameters (Reviews) :

 

business_name - Business Name. Type String.

 

avg.rating - Average Star Rating of Business. Type Double.

 

neighbourhood - Business Neighborhood. Type String. Supports CSV Strings.

 

latitude - Latitude value of busines location. Type Double.

 

longitude - Longitude value of busines location. Type Double.

city - City of Business. Type String.

 

state - State of Business. Type String.

business_id - Business ID. Type Integer.

 

type - Type of Business, defined only as "restaurant", "shop", "hotel", "cafe" and "bar" in String format.

Note: Lenient search outputs can have multiple businesses in JSON Object.


 

Sample Request (Reviews) :

 

/Yap/api?app_id=11&request_type=reviews&business_id=2

 

Request Parameters :

 

app_id - An authentication key provided to developers from the developer console

 

request_type - Provides two APIs; Reviews ("reviews") & Business Search ("search").

 

business_id - Business ID of Business of which the Reviews are required. Type Integer. 


Output Parameters (Reviews) :

 

 

 

date - BDate of Posting Review. Type String. Format YYYY-MM-DD. Example, 2015-11-04.

 

rating - Star Rating of Business. TypeInteger.

 

review - User Review of Business. Type String.

 

latitude - Latitude value of busines location. Type Double.

 

longitude - Longitude value of busines location. Type Double.

 

cool - Number of Cool votes. Type Integer.

 

useful - Number of Useful votes. Type Integer.

 

funny - Number of Funny votes. Type Integer.

 

user_name - User Name of reviewer in String format.

 

Note: Outputs can have multiple reviews in JSON Object.


Sample Outputs (Errors) :

Incorrect app_id.

{"details":"app_id not found","authentication":"failed"}

Incorrect business_id or no reviews yet.

{"reviews":[]}

Incorrect Search Parameters

{"error":"incorrect query parameters"}

No business found.

{"businesses":[]}
